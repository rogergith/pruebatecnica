# pruebatecnica


- DESCRIPCION
	- Este proyecto trata de un APIRest para autenticacion protegido con Json Web Token + Swagger para documentar la API.

	Usamos las seguientes tecnologias:

	- Capa persistencia
		+ MySQL
	- Capa Servidor
		+ SpringBoot
		+ Swagger
		+Json Web Token
	- Capa fron
		+ AngularJS
